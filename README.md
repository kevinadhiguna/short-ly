# Short.ly

Short.ly is a link shortener that helps you monitoring number of clicks for each shortened links.

## Demo

This app also provides the `number of clicks` of the shorthened links alongside `full URL` and its `shorthened link` :<br><br>
<img src="https://raw.githubusercontent.com/kevinadhiguna/short-ly/master/demo/1.png" width="90%"></img>

## How to Run :
1) Install dependencies:<br>
Using yarn :<br>
`yarn install`<br>
Using NPM : <br>
`npm install`<br>
2) Please make sure that MongoDB is running in the background. 
3) Open a terminal or Command Prompt :<br>
Using yarn :<br>
`yarn devStart`<br>
Using NPM :<br>
`npm devStart`<br>
4) The App is running on `localhost:5000`!
